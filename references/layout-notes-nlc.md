# 笔记应用布局规则（NLC 适配）

本文档定义「小米笔记」在 Fold 和 Pad 上的布局适配规则。笔记的一级导航是"文件夹"，在手机/Fold 上以标签栏形态嵌入列表栏顶部，在 Pad 上独立为侧边导航栏（N 栏）。

> **负责人**：唐瀚
> **数据基准**：以 `device-dimensions.md` 为准，本文档仅补充笔记特有的适配逻辑。

---

## 笔记的导航结构映射

| 终端 | 文件夹导航形态 | 布局模型 |
|------|-------------|---------|
| Phone | 标签栏（L 栏顶部） | 单栏 |
| Fold 内屏 | 标签栏（L 栏顶部） | LC 双栏（N 嵌入 L） |
| Pad（w ≥ 960dp） | 侧边导航栏（独立 N 栏） | NLC 三栏并存 |
| Pad（600 ≤ w < 960dp） | 侧边导航栏（覆盖全窗口） | LC + N 浮层覆盖 |

---

## Fold 内屏布局（LC，N 嵌入 L）

Fold 内屏不支持独立 N 栏，文件夹导航以标签栏形式嵌入 L 栏顶部。

### 结构示意

```
┌───────────────────────────────────┐
│            全局状态栏（888dp）       │
├───────────────┬─┬─────────────────┤
│  L 栏 (353dp) │ │  C 栏 (535dp)   │
│ ┌───────────┐ │1│                 │
│ │ 标签栏     │ │p│   笔记编辑器     │
│ │(文件夹导航)│ │x│                 │
│ ├───────────┤ │ │                 │
│ │           │ │ │                 │
│ │  笔记列表  │ │ │                 │
│ │           │ │ ├─────────────────┤
│ │           │ │ │ AI 输入框        │
│ │           │ │ │ (左右各 16dp)    │
└─┴───────────┴─┴─┴─────────────────┘
```

### L 栏内部结构（Fold）

从上到下：
1. **标签栏**（文件夹导航）— 位于 L 栏最顶部，用于切换文件夹
2. **笔记列表** — 填充标签栏以下的剩余空间，垂直滚动

### C 栏内部结构（Fold）

从上到下：
1. **笔记编辑器** — 填充主体区域
2. **AI 输入框** — 固定在 C 栏底部，距 C 栏左右各 16dp

## Pad 布局

### 宽窗口模式（w ≥ 960dp）— NLC 三栏并存

#### 结构示意

### 窄窗口模式（600 ≤ w < 960dp）— N 浮层覆盖

N 栏以浮层形式覆盖整个窗口（遮挡 L 栏），底层仍为 LC 双栏。

行为规则：
- 默认状态：N 栏展开，覆盖整个窗口
- 选择文件夹后：N 栏收起，回到 LC 双栏
- L 与 C 的栏宽按当前窗口宽度下的 LC 规则计算

## 适配执行要点

### Phone → Fold 适配

1. 保留标签栏在 L 栏顶部，不转为独立 N 栏
2. 笔记列表放入 L 栏（353dp），填充标签栏以下空间
3. 笔记编辑器放入 C 栏（535dp）
4. AI 输入框固定在 C 栏底部（左右各 16dp）
5. 删除手机端底部手势条
6. L 与 C 之间加 1px 分割线

### Phone → Pad 适配

1. 标签栏从 L 栏顶部提取，转为独立 N 栏（Sidebar-Component）
2. N 栏默认展开（272dp），NavigationSizeSection = NLC导航
3. 笔记列表放入 L 栏
4. 笔记编辑器放入 C 栏
5. AI 输入框固定在 C 栏底部（左右各 16dp）
6. 删除手机端底部手势条
7. L 与 C 之间加 1px 分割线，N 与 L 之间无分割线

## 元素迁移规则

### 工具入口

| 终端 | 位置 | 备注 |
|------|------|------|
| Phone | 首页右上角 | |
| Fold | L 栏右上角 | 位置不变 |
| Pad | N 栏右上角 | 从 L 栏迁移至 N 栏 |

### 新建笔记工具栏

悬浮工具栏，固定宽度 220dp，水平居中，距离所在栏底部 24dp。

| 终端 | 所在栏 | 水平居中基准 |
|------|--------|-----------|
| Phone | 窗口 | 窗口宽度居中 |
| Fold | L 栏 | L 栏宽度居中 |
| Pad | N 栏 | N 栏宽度居中 |

## 特殊处理

- **空列表态**：L 栏无笔记时，C 栏显示空状态占位（具体样式 TODO）
- **N 栏浮层动画**：窄窗口模式下 N 栏的展开/收起动画由组件自身处理，适配时不需要额外设置
- **横竖屏旋转**：遵循 `device-dimensions.md` 中的横竖屏旋转规则，展开态映射展开态，收起态映射收起态
